# Configuration du cluster MongoDB

## Objectif

Cette étape consiste à préparer l'infrastructure MongoDB distribuée qui servira de base au projet G_4_Pay.

## Réalisations

- Création de l'arborescence du projet.
- Création des scripts de démarrage et d'initialisation du cluster.
- Configuration d'un Config Server.
- Configuration de quatre Shards.
- Configuration du routeur Mongos.
- Initialisation des Replica Sets.
- Ajout des quatre shards au cluster.
- Vérification du bon fonctionnement du cluster.

## Résultat

Le cluster MongoDB est opérationnel et prêt à accueillir la base de données `momo` ainsi que les collections du projet.
